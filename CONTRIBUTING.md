# Contributing to This Project

Hello! Thank you for your interest in this project! We appreciate every contribution, so please feel invited to help out!

## Getting Started

The documentation for getting started is very short. You can find it in the [README file](README.md).

## To Make a Contribution

1. Begin by [making an issue](https://github.com/deweesevs/privacy-for-gradescope/issues). Please use the issue template closest to your contribution:
   - Feature Request
   - Bug Report
   - Task
2. Fork the repository.
3. Clone your fork.
4. Create a branch with a descriptive name and switch to it.
5. Make your changes on that branch.
6. Install **pre-commit** if you have not already.
   ```
   brew install pre-commit
   ```
7. Commit and push your changes.
8. Create a pull request on the main repository.
   - Provide as much detail as possible.
   - All pull requests must include `Closes #X`, where `X` is the issue number from Step 1.
9. Wait for a review and feedback.

Once your work is merged, feel free to delete your fork.
